# MERN Product Dashboard

This is a full-stack MERN CRUD application.

## Tech Stack
- MongoDB
- Express.js
- React.js (Vite)
- Node.js
- Tailwind CSS

## Features
- User Login & Register
- Product CRUD (Create, Read, Update, Delete)
- Dashboard UI
- REST API

## Installation

Backend:
npm install
node server.js

Frontend:
npm install
npm run dev
